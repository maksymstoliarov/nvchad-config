# Prerequisites

1. Neovim

# Installation

2. Clone config
```bash
git clone https://github.com/maksymstoliarov/nvchad-config ~/.config/nvim && nvim
```

3. Install plugins
```bash
:Lazy sync
:MasonInstall
```

4. Load markdown preview

1. :Lazy load markdown-preview.nvim
2. :Lazy build markdown-preview.nvim
