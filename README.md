# Prerequisites

1. Neovim

# Installation

1. Clone config
```bash
git clone https://github.com/maksymstoliarov/nvchad-config ~/.config/nvim && nvim
```

2. Install plugins
```bash
:Lazy sync
:MasonInstall
```

3. Load markdown preview
```
:Lazy load markdown-preview.nvim  
:Lazy build markdown-preview.nvim
```
